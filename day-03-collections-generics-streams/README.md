\# Day 3 – Collections, Generics \& Streams



\## Overview



Day 3 focused on learning how Java manages groups of objects using the Collections Framework, how Generics provide type safety, and how Streams simplify data processing using a functional programming style.



The concepts were applied by building an in-memory Expense Tracker application.
